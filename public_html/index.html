Understanding the Parallax Effect

The parallax effect is a technique used in web design where background images move slower than foreground images, creating an illusion of depth and immersion. It adds a dynamic element to the website and enhances the user experience.

3D CSS Parallax Depth Effect

  • Perspective and Transformations: The effect is achieved by using CSS properties like perspective and transform. perspective sets the depth of the view for the 3D transformations, and transform applies the actual 3D transformations such as rotations and translations.

  • Container Setup: First, a container element is set up with a perspective value that determines the intensity of the 3D effect. This container acts as the perspective origin for all the child elements.

  • Card Setup: Each card or element that needs the parallax effect is styled with transform-style: preserve-3d to ensure that its children maintain their 3D positions relative to the parent.

  • Mouse Interaction: Mouse movement triggers the parallax effect. As the mouse moves, JavaScript calculates the position of the cursor relative to the viewport and adjusts the rotation values of the container and the translation values of the card elements accordingly.

  • Depth Perception: The depth of each card is controlled by applying different translation values along the z-axis. As the mouse moves, cards closer to the viewer move more prominently than those farther away, creating the illusion of depth.

  • Background Parallax: Additionally, background images within the cards can also be animated to move at a different rate, enhancing the perception of depth.

How it works

Type of Code Code Description
HTML The HTML code defines a structure for the cards using <div> elements with appropriate classes.

Each card contains:

(<div class="card__bg">) = a background element

(<img class="card__img">) = an image element

(<div class="card__text">) = a text element

(<p class="card__title">) = and a title element.
JavaScript The JavaScript code handles the 3D parallax effect based on mouse movement. It selects the necessary elements from the DOM using document.querySelector and document.querySelectorAll.

A function calcValue calculates the rotation and translation values based on the mouse position.

Event listeners are added to track mouse movement (mousemove event). Within the event listener, the rotation and translation values are calculated and applied to the .cards, .card__img, and .card__bg elements to achieve the parallax effect.
CSS The CSS codestyles the appearance of the cards and their elements. It sets the background color, border radius, box shadow, padding, perspective, and other properties for the .cards container and individual .card elements.

The .card__img and .card__bg classes define styles for the card images and background respectively.

Specific styles for each card (.card__one, .card__two, .card__three) customize their appearance and positioning.

Transitions and hover effects are also defined to create interactive behavior.
